Instructions

This is a flight insurance App. The list below shows a list of flight you could choose from to get insurance. If the status of the flight is "true", it means it is eligble for insurance. To buy Insurance, you need to have your ether address ready for input. You could use this address to buy insurance, check your balance and manage your outstanding credit all in one click. Please note that for each purchase the insured amount should not be bigger than 1 ether.

**To start testing, please press the setup button at the bottom of the page to start. Once you see the notification. Refresh the page. You can start using the app when all the flight status becomes true
